# airbnb-pricing-analysis
Analyzed Airbnb listing data to identify pricing drivers across NYC neighborhoods, room types, property types, and listing features. Cleaned and modeled 33K+ records in R using EDA, log-price transformation, regression, and tree-based methods to translate market patterns into actionable insights.
